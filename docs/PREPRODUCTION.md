- Одиночная игра

# 2. Заморозка игрового дизайна

## Жанр
2D Action Platformer

## Основная концепция
Игрок управляет героем-изгоем с демоном внутри.  
Геймплей строится на балансе между контролем и силой.

## Основные механики
- Передвижение (бег, прыжок, гравитация)
- Боевая система (атаки, урон, i-frames)
- Demon System (метр демона + трансформация)

## Структура игры
INTRO → LEVEL 1 → LEVEL 2 → BOSS 1 → CUTSCENE → LEVEL 3 → LEVEL 4 → BOSS 2 → LEVEL 5 → FINAL BOSS → ENDING

## Враги
- Наземные
- Летающие
- Стреляющие
- Мини-боссы

## Боссы
- Босс 1 (базовый)
- Босс 2 (несколько фаз)
- Финальный босс (встречается в начале и в конце)
